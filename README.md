# 🚀 Nexus AI – Advanced Recruitment Intelligence System
## Overview
Nexus AI is an AI-powered recruitment system designed to automate resume screening using advanced Natural Language Processing (NLP) techniques. It leverages BERT (Sentence-Transformers) to perform semantic matching between resumes and job descriptions, enabling accurate candidate ranking beyond traditional keyword-based methods.
## 🔥 Key Features
### 🧠 Semantic Similarity Engine
Uses BERT embeddings to understand contextual meaning of resumes and job descriptions
### 📄 Automated Resume Parsing
Extracts text from PDFs using PyMuPDF
### 🧹 NLP Data Pipeline
Performs preprocessing using Regex, cleaning and structuring raw data
### 🎯 Candidate Ranking System
Uses cosine similarity to rank candidates based on relevance
### 📊 Skill Gap Identification
Identifies missing technical skills from candidate profiles for better hiring insights

## 🛠️ Tech Stack
### 1.Extract resume text from PDFs
### 2.Clean and preprocess data using NLP techniques
### 3.Generate embeddings using BERT
### 4.Compute similarity between resume and job description
### 5.Rank candidates based on similarity scores
### 6.Identify missing skills for improvement insights

## 📊 Use Case
### Automates resume screening
### Reduces manual hiring effort
### Improves candidate-job matching accuracy
### Provides actionable insights for recruiters
